# A5: Bump version in marketplace.json

**Avoid:** Don't change any field other than `version`. The version must be consistent across all three plugins.

## Steps

### Step 1: Read current marketplace.json

Read `.claude-plugin/marketplace.json` and confirm current version is `1.12.0` across all three plugins.

### Step 2: Bump version to 1.13.0

Update all three `version` fields from `1.12.0` to `1.13.0` using jq:

```bash
jq '(.plugins[].version) = "1.13.0"' .claude-plugin/marketplace.json > tmp.$$.json && mv tmp.$$.json .claude-plugin/marketplace.json
```

### Step 3: Verify

```bash
jq -r '.plugins[].version' .claude-plugin/marketplace.json | sort -u
```

Should output exactly one line: `1.13.0`.

### Step 4: Commit

Commit the version bump.
