# A1: Create defaults.json schema

**Avoid:** Don't put defaults.json inside `.claude-plugin/` or `scripts/`. It lives at the repo root because `CLAUDE_PLUGIN_ROOT` points there, and the script reads it as `${CLAUDE_PLUGIN_ROOT}/defaults.json`. Don't add settings beyond the 8 specified in the design doc.

## Steps

### Step 1: Create `defaults.json` at the repo root

Create the file with all 8 settings. Each setting has `type`, `default`, `description`, and `used_by` fields. Enum types also have a `values` array.

```json
{
  "skip_tests": {
    "type": "bool",
    "default": false,
    "description": "Skip test suite before committing",
    "used_by": ["pr-create"]
  },
  "review_mode": {
    "type": "enum",
    "values": ["automated", "deliberate"],
    "default": "deliberate",
    "description": "Default PR review mode",
    "used_by": ["pr-review"]
  },
  "skip_review": {
    "type": "bool",
    "default": false,
    "description": "Skip fresh-eyes subagent review dispatch",
    "used_by": ["pr-review"]
  },
  "merge_strategy": {
    "type": "enum",
    "values": ["squash", "rebase"],
    "default": "squash",
    "description": "Default merge strategy for standard PRs (integration branches and phase PRs retain auto-detected strategy)",
    "used_by": ["pr-merge"]
  },
  "workflow": {
    "type": "enum",
    "values": ["pr-create", "pr-merge", "plan-only"],
    "default": "pr-create",
    "description": "Default post-orchestration workflow",
    "used_by": ["design", "orchestrate"]
  },
  "execution_mode": {
    "type": "enum",
    "values": ["subagents", "agent-teams"],
    "default": "subagents",
    "description": "Default execution mode for plan dispatch",
    "used_by": ["design", "orchestrate"]
  },
  "review_wait_minutes": {
    "type": "int",
    "default": 10,
    "description": "Poll timeout in minutes for external review bot comments (warm-up periods remain fixed)",
    "used_by": ["orchestrate", "pr-review"]
  },
  "re_review_threshold": {
    "type": "int",
    "default": 5,
    "description": "Issue count above which reviewer is re-dispatched after fixes",
    "used_by": ["design", "orchestrate", "design-review", "plan-review", "implementation-review"]
  }
}
```

### Step 2: Verify structure

```bash
jq '.skip_tests.type' defaults.json
jq '.review_mode.values' defaults.json
jq '. | keys | length' defaults.json
```

Expected: `"bool"`, `["automated","deliberate"]`, `8`.

Commit: `feat: add defaults.json with 8 settings schema`
