# A6: Integrate review_mode, skip_review, and review_wait_minutes into pr-review

**Avoid:** Don't change the behavior when `--automated`/`-A` is explicitly passed — that flag is tier 1 and always wins. Don't change the `--skip-review`/`-R` flag behavior. Don't change the 90-second and 60-second warm-up periods — those are fixed (the design doc says "warm-up periods remain fixed"). Only the poll timeout (currently hardcoded at 10 minutes) is configurable via `review_wait_minutes`.

## Steps

### Step 1: Add review_mode fallback to Step 2 (Mode Selection) in `skills/pr-review/SKILL.md`

Find the current Step 2 opening:

```markdown
### Step 2: Mode Selection

If `--automated`/`-A` flag was passed, use automated mode (skip prompt). If both `--automated` and `--skip-fixes` are passed, fail fast — these flags are mutually exclusive.

Otherwise, AskUserQuestion:
```

Replace with:

```markdown
### Step 2: Mode Selection

If `--automated`/`-A` flag was passed, use automated mode (skip prompt). If both `--automated` and `--skip-fixes` are passed, fail fast — these flags are mutually exclusive.

If no mode flag was passed, check `${CLAUDE_PLUGIN_ROOT}/scripts/caliper-settings get review_mode`. If it returns `automated`, use automated mode (skip prompt). If `deliberate` (or any other value), proceed to the prompt below.

Otherwise, AskUserQuestion:
```

### Step 2: Add skip_review fallback to Step 4 (Dispatch Subagent) in `skills/pr-review/SKILL.md`

Find the current Step 4 opening:

```markdown
### Step 4: Dispatch Subagent in Background

Skip if `--skip-review` was passed.
```

Replace with:

```markdown
### Step 4: Dispatch Subagent in Background

Skip if `--skip-review` was passed. If `--skip-review` was not passed, check `${CLAUDE_PLUGIN_ROOT}/scripts/caliper-settings get skip_review` — if it returns `true`, skip this step.
```

### Step 3: Add review_wait_minutes to Step 5 (External Feedback) in `skills/pr-review/SKILL.md`

Find the timeout line in the user-selected automated / deliberate polling section:

```markdown
6. **Timeout:** 10 minutes max. Proceed with available feedback.
```

Replace with:

```markdown
6. **Timeout:** `${CLAUDE_PLUGIN_ROOT}/scripts/caliper-settings get review_wait_minutes` minutes max (default: 10). Proceed with available feedback.
```
